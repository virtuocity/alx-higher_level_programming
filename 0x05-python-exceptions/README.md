# 0x05. Python - Exceptions
## Notes and Thoughts
There are (at least) two distinguishable kinds of errors: syntax errors and exceptions.  
*Syntax Errors* Syntax errors, also known as parsing errors, are perhaps the most common kind of complaint you get while you are still learning Python  
 Exceptions come in different types, and the type is printed as part of the message: the types in the example are ZeroDivisionError, NameError and  
 TypeError
